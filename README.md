## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## GitHub Pages Deployment

1. Build the project:
   ```bash
   npm run build
   ```

2. The `dist` folder contains the static files ready for deployment.

3. Configure GitHub Pages to serve from the `dist` folder or push the contents of `dist` to the `gh-pages` branch.

4. Update the `base` path in `vite.config.js` to match your repository name if different from `/Wedding_web/`.

## Customization

Update the following in the components:
- Couple names
- Wedding date
- Venue locations
- Registry links
- Menu items
- Photo sharing links
